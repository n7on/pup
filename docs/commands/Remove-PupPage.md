# Remove-PupPage

## SYNOPSIS
Closes a browser page.

## SYNTAX

## DESCRIPTION
Closes the specified page (tab) in the browser.
The page object should not be used after removal.

## EXAMPLES

### Example 1: Close a page
```
Remove-PupPage -Page $page
```

Closes the browser tab.
